# Streamlit UI Components Demo App

This application demonstrates the use of various Streamlit UI components.

## Features
- Uses 25+ Streamlit UI components
- Includes sidebar navigation
- Student input form
- Dashboard view
- About section explaining the app

## Target Users
Students learning how Streamlit interfaces work.

## Inputs Collected
Name, age, gender, course, skills, study hours, and more.

## Outputs
Displays submitted user information and example dashboard metrics.
